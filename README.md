# Sistema de Cursos (Java)

Este é um projeto simples desenvolvido em Java puro com o objetivo de consolidar minha base na linguagem antes de avançar para frameworks como Spring Boot.

##  Objetivo

Praticar conceitos fundamentais do Java e fortalecer o raciocínio lógico voltado para desenvolvimento backend.

##  O que foi aplicado

- Programação Orientada a Objetos (POO)
- Collections (`List`)
- API de Streams
- `Optional`
- Estrutura de camadas (Model / Service)
- CRUD completo (Create, Read, Update, Delete)
- Manipulação de dados em memória
- Entrada de dados via console (`Scanner`)

##  Funcionalidades

- Cadastro, listagem, busca, atualização e remoção de usuários
- Cadastro, listagem, busca, atualização e remoção de cursos
- Associação de usuários a cursos
- Consulta de usuários matriculados em um curso

##  Sobre o projeto

Este projeto foi desenvolvido como parte do meu processo de aprendizado em Java.  
Apesar de simples, ele representa um avanço importante na minha compreensão de:

- Estruturação de código
- Separação de responsabilidades
- Lógica de negócios
- Uso de recursos modernos da linguagem

##  Próximo passo

Evoluir este projeto utilizando Spring Boot, transformando a aplicação em uma API REST.
